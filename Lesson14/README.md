三角函数
========

## 知识点

numpy提供了三角函数的计算功能

* sin
* cos
* tan

## 实战演习

~~~python
import numpy as np

print("----- sin -----")
val = np.sin(np.pi / 6)
print(val)
print(np.round(val, 1))

print("----- cos -----")
val = np.cos(np.pi / 3)
print(val)
print(np.round(val, 1))

print("----- tan -----")
val = np.tan(np.pi / 4)
print(val)
print(np.round(val, 1))
~~~
